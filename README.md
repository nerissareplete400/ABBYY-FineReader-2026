Select the Scan to Word or Scan to PDF button. The program looks for connected scanners or allows you to select a saved image file from your hard drive. It detects the text during the scanning process and saves it as a document you can edit.

### Editing PDF Files
You open a PDF by dragging it into the workspace. The software recognizes the text blocks and images. You click on any part of the text to delete, correct, or add new words. You drag images around the page to change the layout.

### Searching Within Documents
The software creates a layer of searchable text on top of images. Use the find tool to search for specific words or phrases across a large batch of documents. This saves time when you work with extensive archives.
